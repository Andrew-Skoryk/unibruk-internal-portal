# Unibruk Internal Portal

Внутрішній портал для відділу збуту Unibruk. Проєкт зосереджений на швидкому доступі до контактів, фільтрації, систем знижок і довідкових даних.

## Що всередині

- **Головна панель** з ключовими блоками та статистикою
- **Контакти** з фільтрами: пошук, менеджер, статус, дата наступного контакту, загальна сума
- **Система знижок** (фізичні особи та укладальники/дилери)
- **Скелет‑лоадери** для кращого UX під час завантаження
- **UI компоненти** на базі shadcn/ui

## Стек

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui

## Локальний запуск

```bash
npm install
npm run dev
```

Відкрити: `http://localhost:3000`

## Структура сторінок

- `/` — Головна
- `/clients` — Контакти та фільтрація
- `/discounts` — Система знижок

## Деплой

Проєкт готовий до деплою у Vercel (через GitHub інтеграцію).
